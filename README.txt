BuZZ - социальная сеть. В ней есть:
• Авторизация пользователей, работа с куки
• Страницы пользователей с постами
• Редактирование своей страницы(добавление фотографии на аву, создание/удаление постов)
• Подписчики и друзья
• Поиск пользователей в соц сети
• Сообщения
• Новостная лента

Реализованные технологии:
• Работа с API(можно получить список новостей по адресу '/json_news')
• Работа с SQL
